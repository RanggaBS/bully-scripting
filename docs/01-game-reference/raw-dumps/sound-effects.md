# Sound Effects

Sound effects are pre-recorded audio assets used throughout the game to provide auditory feedback for actions, events, and ambient environments. These sounds are stored in `.bnk` (sound bank) files.

In scripting, you can play any of these sound effects using functions like [`SoundPlay2D`](/docs/game-reference/global-functions/SoundPlay2D) for non-spatial audio or [`SoundPlay3D`](/docs/game-reference/global-functions/SoundPlay3D) for audio that originates from a specific point in the game world.

## Example Usage

To use a sound effect, you must first ensure its corresponding sound bank (`.bnk` file) is loaded using [`SoundLoadBank`](/docs/game-reference/global-functions/SoundLoadBank).

### Playing a 2D Sound

A 2D sound is played without any spatial positioning, making it ideal for UI feedback or general notifications.

```lua
-- Load the sound bank containing the desired effect
SoundLoadBank('Hud.bnk')

-- Play the 'BuyItem' sound effect
SoundPlay2D('BuyItem')
```

### Playing a 3D Sound

A 3D sound is emitted from a specific coordinate in the world, allowing for spatial audio that changes based on the player's position and orientation relative to the source.

```lua
-- Load the sound bank
SoundLoadBank('Crane.bnk')

-- Get the player's current position
local x, y, z = PlayerGetPosXYZ()

-- Play the 'CRANE' sound effect at the player's location
SoundPlay3D(x, y, z, 'CRANE')
```
